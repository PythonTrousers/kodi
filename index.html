⚙️ How to Install in Kodi

  1. Open Kodi and navigate to the Settings gear icon.
  2. Open File Manager and select Add source.
  3. Click on <None> and enter the URL exactly:
    https://pythontrousers.github.io/kodi/
  4. Name the media source PythonTrousers and click OK.
  5. Return to the home screen, click Add-ons, and click the open box icon at the top left.
  6. Select Install from zip file.
  7. Select PythonTrousers and then select the repository.pythontrousers-0.2.0.zip file.
